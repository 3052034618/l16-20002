# 艺术品管理与展览调度平台 产品需求文档

## 1. 产品概述

艺术品管理与展览调度平台是面向美术馆、画廊及艺术机构的一站式运营管理系统，整合藏品入库、展览策划、销售租赁、布展任务、环境监控及运输保险六大核心业务模块，通过智能化提升艺术资产管理与运营效率，实现艺术品全生命周期数字化管理。

- **目标用户**：美术馆馆长、策展人、藏品保管员、艺术家、财务人员
- **核心价值**：智能估值、冲突检测、自动调度、实时监控、多级审批、数据可视化

## 2. 核心功能

### 2.1 用户角色

| 角色 | 权限等级 | 核心权限 |
|------|----------|----------|
| 艺术家 | 一级 | 查看本人作品信息、展览记录、销售记录 |
| 策展人 | 二级 | 展览策划、布展方案、环境监控、展览任务管理 |
| 保管员 | 三级 | 藏品入库出库、库存管理、价值评估、运输保险管理 |
| 馆长 | 四级（全局） | 所有功能、审批、估值规则调整、全局数据看板 |

### 2.2 功能模块

1. **首页大屏**：实时数据看板、展厅热度、环境达标率、布展进度、运输在途、数据筛选
2. **藏品管理**：藏品入库、数字指纹生成、自动估值、藏品列表、藏品详情
3. **展览策划**：展览创建、智能布展推荐、冲突检测、展览列表
4. **销售租赁**：申请提交、三级审批、越级机制、销售/租赁记录
5. **布展任务**：任务分配、工种排班、完工确认、进度追踪
6. **环境监控**：实时监测、超标告警、设备调节、维修工单
7. **运输保险**：物流推荐、在途追踪、保险预警、续保审批
8. **系统管理**：用户权限、估值规则、数据导出

## 3. 核心流程

### 3.1 藏品入库流程

保管员录入作品基本信息 → 系统自动拍照生成数字指纹 → 系统根据艺术家声望和成交记录自动预估价值区间 → 馆长审核确认 → 入库完成

```mermaid
flowchart LR
    A["保管员录入信息"] --> B["生成数字指纹"]
    B --> C["自动价值预估"]
    C --> D["馆长审核"]
    D -->|通过 --> E["入库完成"]
    D -->|驳回 --> F["修改信息"]
    F --> C
```

### 3.2 展览策划流程

策展人创建展览 → 选择作品 → 系统智能推荐布展方案 → 冲突检测 → 调整确认 → 生成布展任务

```mermaid
flowchart LR
    A["创建展览"] --> B["选择作品"]
    B --> C["智能布展推荐"]
    C --> D["冲突检测"]
    D -->|无冲突 --> E["确认方案"]
    D -->|有冲突 --> F["调整方案"]
    F --> C
    E --> G["生成布展任务"]
```

### 3.3 销售租赁审批流程

申请人提交申请 → 馆长审批 → 委员会审批 → 财务审批 → 审批完成 → 出库/发货

```mermaid
flowchart LR
    A["提交申请"] --> B["馆长审批"]
    B -->|超48h --> C["自动越级"]
    B -->|通过 --> D["委员会审批"]
    C --> D
    D -->|超48h --> E["自动越级"]
    D -->|通过 --> F["财务审批"]
    E --> F
    F -->|通过 --> G["审批完成"]
    G --> H["出库/发货"]
```

### 3.4 环境监控告警流程

传感器实时监测 → 数据超标 → 自动调节设备 → 生成维修工单 → 超30分钟未达标 → 自动升级

```mermaid
flowchart LR
    A["实时监测"] --> B{"数据超标?"}
    B -->|是 --> C["自动调节设备"]
    C --> D["生成维修工单"]
    D --> E{"30分钟内达标?"}
    E -->|否 --> F["自动升级告警"]
    E -->|是 --> G["告警解除"]
    B -->|否 --> A
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深墨色（#1a1a2e）+ 金色点缀（#d4af37），营造美术馆典雅高端氛围
- **辅助色**：象牙白（#f5f0e8）背景、深灰文字、品牌一致性
- **按钮风格**：圆角矩形、金色边框、悬停金色填充、优雅过渡动画
- **字体**：标题使用衬线字体（Playfair Display），正文使用无衬线字体（Noto Sans SC）
- **布局风格**：卡片式布局、大留白、精致边框、阴影层次
- **图标风格**：线性图标、金色描边、简约优雅
- **整体调性**：博物馆级别的典雅感、数字化科技感、专业可信赖

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页大屏 | 数据看板 | 深色背景、金色数据卡片、实时数据刷新动效、环形进度图、热力图 |
| 首页大屏 | 筛选栏 | 顶部筛选区、下拉选择器、导出按钮、时间范围选择 |
| 藏品管理 | 列表页 | 网格/列表双视图、搜索栏、筛选条件、藏品卡片（缩略图+信息） |
| 藏品管理 | 入库表单 | 分步骤表单、图片上传区、自动指纹生成动画、价值预估展示 |
| 展览策划 | 展览列表 | 时间轴视图、展览卡片、状态标签 |
| 展览策划 | 布展方案 | 展厅平面图、作品拖拽布局、冲突提示、推荐标记 |
| 销售租赁 | 审批列表 | 审批流时间线、状态徽章、加急标记 |
| 布展任务 | 任务看板 | 看板列（待分配/进行中/已完成）、任务卡片、拖拽排序 |
| 环境监控 | 监控面板 | 仪表盘样式、实时数据波形、告警闪烁效果、设备状态 |
| 运输保险 | 在途追踪 | 地图风格、运输节点、时间线 |

### 4.3 响应式设计

- 桌面端优先设计，适配 1920px 及以上大屏展示
- 首页大屏针对大尺寸显示器优化，支持全屏模式
- 平板端自适应布局，侧边栏可折叠
- 移动端保留核心功能，简化操作流程

### 4.4 动效与交互

- 数据数字滚动动画，数字变化时平滑过渡
- 卡片悬停微上浮效果，金色边框发光
- 页面切换淡入淡出，内容区域有层次入场
- 告警状态脉冲闪烁效果，吸引注意力
- 进度条平滑填充动画
- 筛选条件展开收起平滑过渡
